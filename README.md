O **StockHub** é um sistema moderno, rápido e responsivo desenvolvido para facilitar o controle de estoque, produção e logística da **Stef Doces**. Construído com tecnologias web de ponta, o sistema funciona perfeitamente tanto no computador quanto no celular (como um aplicativo nativo - PWA).

🌐 **[Acesse o sistema online aqui!](https://stehlima.github.io/gestao-de-estoque/)**

---

## ✨ Funcionalidades Principais

* **📊 Dashboard Analítico:** Acompanhamento financeiro diário e gráficos dinâmicos de vendas utilizando `Recharts`.
* **📦 Gestão de Estoque:** Controle rígido da quantidade de insumos, com alertas automáticos de vencimento e estoque crítico.
* **🧁 Fichas Técnicas de Produção:** Cadastro de receitas interligadas com os ingredientes em estoque.
* **🔔 Central de Notificações:** Avisos automáticos em tempo real sobre entregas, estoques baixos e novidades.
* **🏢 Gestão de Fornecedores:** Lista de parceiros de negócios com funcionalidade para gerar e simular envios de pedidos de reposição com um clique.
* **⏱️ Auditoria e Histórico:** Rastreamento completo de movimentações (entradas e saídas de produtos), informando *quem* fez e *quando*.
* **📱 Progressive Web App (PWA):** O site pode ser instalado no smartphone, permitindo abrir sem barras de navegador e até suporte offline.
* **🌙 Dark Mode & Idiomas:** Suporte nativo para troca de temas (Claro/Escuro) e idiomas (PT/EN).

---

## 🚀 Tecnologias Utilizadas

* **[React 19](https://react.dev/)**: Biblioteca JavaScript para construção da interface.
* **[Vite](https://vitejs.dev/)**: Ferramenta de build de altíssima velocidade.
* **[Lucide-React](https://lucide.dev/)**: Biblioteca de ícones modernos e leves.
* **[Recharts](https://recharts.org/)**: Biblioteca para criação de gráficos iterativos.
* **[Vite-Plugin-PWA](https://vite-pwa-org.netlify.app/)**: Configuração para transformar o site em aplicativo offline.
* **Vanilla CSS**: Estilização totalmente customizada com variáveis globais (Glassmorphism + Dark Mode).
